# students.webappKurs2026

teknikkurs för ungdomar 2026

Web apps project based course that use public APIs through AI-assisted "vibe coding," while helping students to genuinely understand the code they ship.

By the end, every student has a **live, deployed web app** of their own design
that talks to at least one public API.
- **Format:** 7 days × 2 hours (14 h total)
- **Approach:** concept-first + AI as a coding partner (not a crutch)
- **Full lesson plan:** [`lectures/plan.md`](lectures/plan.md)
## Course arc
| Day | Theme |
|-----|-------|
| 1 | AI intro, how the web works, first vibe-coded page |
| 2 | Prompting well, reading AI-generated code |
| 3 | APIs, JSON, first `fetch` from a public API |
| 4 | Search-driven app with user input |
| 5 | Git, GitHub & basic cyber-security |
| 6 | Project day — design & build your own app |
| 7 | Polish, deploy & showcase |
